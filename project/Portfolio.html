Tech stack · For the technical reader

Under the hood.

No proprietary lock-in. No cookie-cutter themes. The whole stack is open, owned by you, and chosen for the job — not the trend.

01
Frontend
  • Astro — content sites, default for Starter/Business
  • Next.js 15 — when admin panels or auth are needed
  • React 19 — for interactive surfaces
  • Tailwind CSS 4 — design tokens to code, no CSS bloat
  • Framer Motion · Lenis — motion when it earns its place
02
Backend & data
  • Node.js · Vercel Functions · Cloudflare Workers — serverless first
  • Supabase (Postgres) — default DB for products + admin panels
  • Sanity / Contentful — headless CMS when client team will publish
  • Resend · Formspree — transactional email + form submissions
  • Clerk · Auth.js · Supabase Auth — pick by use case
03
Payments & commerce
  • Razorpay — India-first; UPI, cards, NEFT, COD
  • Stripe — international transactions
  • WhatsApp Business API — via wati.io / Twilio
  • Custom storefronts — when Shopify limits matter
  • Shopify Hydrogen / Medusa — for high-volume e-commerce
04
Hosting & infra
  • Vercel — default deploy target; free tier covers SMEs year 1
  • Cloudflare Pages + R2 — when bandwidth or edge matters
  • Netlify — fallback / migration target
  • Cloudflare DNS · CDN · WAF — DDoS / cache layer
  • Domain — owned by client (GoDaddy / Hostinger guidance)
05
SEO · GEO · AI
  • Schema.org JSON-LD — LocalBusiness, FAQPage, Review, Person
  • llms.txt — AI-search discoverability (Claude / GPT / Perplexity)
  • Sitemap.xml + robots.txt — Google Search Console-ready
  • Plausible · Umami — privacy-first analytics, no cookie banner
  • Lighthouse CI — 90+ score gate before launch
06
AI tooling (in product work)
  • Claude · GPT · Gemini — for AI features in client products
  • Anthropic SDK / OpenAI SDK — direct API integrations
  • Vector DB — Pinecone / Supabase pgvector for retrieval
  • WebSeek.ai — own AI builder, eats own dog food
  • Cursor · Claude Code — daily dev tooling

Why custom over Wix / WordPress / Shopify themes? Themes load 1–4 MB of bloat per page, lock you into proprietary editors, and rank poorly. Custom-coded sites load <200 KB, score 90+ on Lighthouse out of the box, and stay yours forever. The tradeoff is you can't drag-and-drop content yourself — which is why Band B onwards ships with a real CMS. Live performance metrics for this exact page are in the footer.
