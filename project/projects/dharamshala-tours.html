Dharamshala Tours

—Three school friends in Kangra, one WhatsApp-first booking flow. We skipped the accommodation-listing pattern the competition uses — the site is part of ~1,000 travel packages booked every month.

Open live site ↗ Build something similar
The brief

Every other Kangra travel operator runs the same template — a list of hotels with prices, a contact form nobody fills. The founders wanted to be different: they sell packages, not rooms, and most of their conversion happens on WhatsApp.

So we built around the WhatsApp moment. Each package page ends in a one-tap WhatsApp deep link with the package pre-filled. The CMS lets the team push a new monsoon or winter itinerary without waiting on a developer. ~1,000 packages are booked through the site each month.

  • ConversionWhatsApp deep-links
  • Volume~1,000 / month
  • EditorSelf-serve CMS
